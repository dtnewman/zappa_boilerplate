About this site

  • Runs on AWS Lambda + API Gateway (there is no permanent server).
  • Built with the Flask framework and deployed using the Zappa package.
  • The backend uses a PostgreSQL database to store information.
  • This site has a few pages (Home, About, etc.) and also features the ability to register and login as a user. I intentionally kept it simple so that this project can be used as a basis for your own projects!
  • Checkout the project on github. Feel free to use it and modify as you see fit. Contributions to the project are always welcome.
